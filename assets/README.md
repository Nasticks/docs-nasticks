# Assets

Ce dossier contient tous les assets statiques du projet.

## Structure

```
assets/
├── images/          # Images pour les articles et pages
│   └── featured/    # Images featured des articles
├── logos/           # Logos et icônes
└── README.md        # Ce fichier
```

## Organisation

- **images/** : Toutes les images utilisées dans les articles MDX
- **logos/** : Logos du site (light/dark mode)
- **featured/** : Images de couverture pour les articles

## Bonnes pratiques

- Utiliser des noms de fichiers descriptifs
- Optimiser les images avant commit (WebP, compression)
- Respecter la structure de dossiers
