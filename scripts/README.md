# Scripts

Ce dossier contient les scripts utilitaires pour la gestion du projet.

## Scripts disponibles

### `blogs.sh`
Script de setup pour créer la structure de blog DevOps & Cloud.
Usage:
```bash
./scripts/blogs.sh
```

### `fichier.sh`
Script utilitaire pour la réorganisation des fichiers de blog.
Usage:
```bash
./scripts/fichier.sh
```

## Bonnes pratiques

- Tous les scripts doivent être exécutables (`chmod +x`)
- Ajouter des commentaires explicatifs
- Utiliser `#!/bin/bash` en en-tête
- Gérer les erreurs avec `set -e` si approprié
